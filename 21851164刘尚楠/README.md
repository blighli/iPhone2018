## Calendar

基于Object-C实现一个简单的Calendar。

### Usage:

```shell
使用:
  命令:
    cal
    cal -m [month]
    cal [month] [year]
    cal [year]
  检查:
    年: 正数
    月: 1-12之间的正数
```

### Make

```shell
cd Calendar
make
./cal [option]
```
